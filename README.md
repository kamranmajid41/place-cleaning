# place-cleaning

A simple flask app built to clean data. 

## Running the app

Clone the repo: 
```
git clone https://github.com/kamranmajid41/place-cleaning.git
```

Navigate to the project directory: 
```
cd place-cleaning
```

Install the required files: 
```
pip3 install -r requirements.txt
```

Run the app 
``` 
flask run 
```
